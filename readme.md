# 📋 Google Sheets байланыстыру нұсқаулығы

## Қадам 1: Google Таблица дайындау

1. [Google Sheets](https://sheets.google.com) сайтына кіріңіз
2. **Жаңа таблица** жасаңыз (мысалы: "Той қонақтары")
3. Бірінші жолға (тақырыптар) мынаны жазыңыз:
   - **A1:** `Аты-жөні`
   - **B1:** `Қатысу жауабы`
   - **C1:** `Уақыты`
   - **D1:** `Серверлік уақыт`

---

## Қадам 2: Apps Script орнату

1. Google Таблицада жоғарғы мәзірден: **Extensions → Apps Script** басыңыз
2. Ашылған редакторда бар кодты **толығымен өшіріңіз**
3. `google-apps-script.js` файлындағы кодты **толығымен көшіріп, қойыңыз**
4. Жобаға ат беріңіз (мысалы: "ТойШақыру")
5. **💾 Сақтау** батырмасын басыңыз (Ctrl + S)

---

## Қадам 3: Web App ретінде жариялау (Deploy)

1. Apps Script редакторында жоғарғы оң жақтағы **Deploy → New deployment** басыңыз
2. ⚙️ таңбасын (Select type) басып, **Web app** таңдаңыз
3. Параметрлерді келесідей толтырыңыз:
   - **Description:** `Той шақыру формасы`
   - **Execute as:** `Me` (өзіңіздің аккаунтыңыз)
   - **Who has access:** `Anyone` (маңызды! Кез-келген адам жібере алатындай)
4. **Deploy** батырмасын басыңыз
5. Егер рұқсат сұраса → **Authorize access** → Аккаунтыңызды таңдаңыз → **Advanced** → **Go to ... (unsafe)** → **Allow**
6. **Web app URL** сілтемесін **көшіріңіз** — ол мынадай болады:
   ```
   https://script.google.com/macros/s/AKfycbx.../exec
   ```

---

## Қадам 4: Сайтпен байланыстыру

1. `index.html` файлын ашыңыз
2. Іздеңіз: `YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE`
3. Оны **өзіңіздің Web App URL сілтемеңізге** ауыстырыңыз:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbx.../exec';
   ```
4. Файлды сақтаңыз

---

## Қадам 5: Тестілеу

1. Сайтты браузерде ашыңыз
2. RSVP формасына тест деректерін жазыңыз
3. "Жіберу" батырмасын басыңыз
4. Google Таблицаны тексеріңіз — деректер пайда болуы тиіс!

---

## 🎵 Фондық музыка қосу

1. `.mp3` форматындағы музыка файлын `index.html` файлымен бір папкаға қойыңыз
2. `index.html` файлындағы `<source src="">` жолын табыңыз
3. Музыка файлыңыздың атын жазыңыз:
   ```html
   <source src="wedding-music.mp3" type="audio/mpeg">
   ```

---

## ⚠️ Маңызды ескертулер

- Google Apps Script-ті **қайта жариялау** (Manage deployments → Edit → Version: New version → Deploy) қажет болса, URL өзгермейді
- `mode: 'no-cors'` қолданылғандықтан, fetch жауабы оқылмайды, бірақ деректер серверге жетеді
- Таблицаның бірінші парағы (Sheet1) қолданылады — пайдалану кезінде белсенді парақ дұрыс екенін тексеріңіз
